# Android Smart MP3 Player

Android application for managing and playing MP3 files with voice control support.

## Overview
This project demonstrates the development of a mobile media application that allows users to organize, play, and control audio files using voice commands.
It also supports cloud storage integration for uploading and sharing audio files.

## Key Features
- MP3 file playback and album organization
- Voice-controlled actions
- Firebase cloud integration for audio upload
- Offline access to stored audio
- Android application built using Java

## Tech Stack
- Java (Android)
- Firebase
- Speech Recognition APIs

## Purpose
This project was developed as part of applied coursework to demonstrate mobile development, cloud integration, and voice control features.

> Note: This project represents educational application development experience.
